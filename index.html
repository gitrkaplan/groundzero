GRID
USAGE

All columns must be used with a wrapping div class="container" and within that container must be a
div class="row". Within that row can contain any number of columns equal to the twelve column grid system. Simply include the number of columns you would like to add followed by -col

EXAMPLE:

<div class="container">
  <div class="row">
    <div class="twelve-col"></div>
  </div>
</div>

BUTTONS
STANDARD
USAGE

Use the standard <button> element with the btn class and any of the following class attributes to create a button.

EXAMPLE:

Hollow
<button class="btn btn-primary-hollow">Primary</button>


Filled
<button class="btn btn-primary">Primary</button>

SQUARE
USAGE

Add the square class to remove the rounded edges.

EXAMPLE:

Hollow
<button class="btn btn-primary-hollow square">Primary</button>


Filled
<button class="btn btn-primary square">Primary</button>

PILL
USAGE

Add the pill class to create a pill shaped button.

EXAMPLE:

Hollow
<button class="btn btn-primary-hollow pill">Primary</button>


Filled
<button class="btn btn-primary pill">Primary</button>

BUTTON WITH ICON
USAGE

Add the icon of choice by using the<i> tag inside of the <button> tag.

EXAMPLE:

Hollow
<button class="btn btn-primary-hollow pill"><i class="fa fa-download" aria-hidden="true"></i>Download</button>


Filled
<button class="btn btn-primary pill"><i class="fa fa-download" aria-hidden="true"></i>Download</button>

NOTE: The icons being used in these examples are from Font Awesome.

CODE
USAGE

There are two different types of code that can be written. Inline code for defining a small snippet such as <p>Lorem ipsum dolor...</p> and blocks of code shown below.

EXAMPLE:

For both the inline code and code block, they will require the code tag and require the class of single .

For the block of code, it will require the pre tag to be the parent of the code tag without the class of single .

Inline
<code class="single">inline code goes here</code>


Code Block
<pre>
  <code>
    const greet = (name) => {
      console.log('Hi, ' + name)
    }
    greet('John Smith')
    // => 'Hi, John Smith'
  </code>
</pre>

TABLES
USAGE

Use the standard <table> tag with the child <tr> tag for a table row and nested <th> tag for the table heading and a nested <td> tag for the table data. On the <td> tag, use <class="primary"> or any other color class attributes to add a color to the table heading.

EXAMPLE:

<table>
  <tr>
    <th>Company</th>
    <th>Contact</th>
    <th>Country</th>
  </tr>
  <tr>
    <td>Alfreds Futterkiste</td>
    <td>Maria Anders</td>
    <td>Germany</td>
  </tr>
</table>
